Console seguro de senha

Este console existe para troca de senha do proprio usuario e reset administrativo controlado. Ele usa a mesma API da producao e invalida sessoes antigas apos mudanca de credencial.

Entre com sua conta para trocar a propria senha. Reset de outra conta aparece apenas para PLATFORM_ADMIN.

Sessao atual

Nenhuma sessao ativa.

Entrar

A sessao fica apenas no navegador atual e nao e gravada fora do sessionStorage.

Trocar minha senha

Depois da troca, os refresh tokens antigos sao revogados e sera necessario entrar novamente.

Reset administrativo

Fluxo restrito a PLATFORM_ADMIN. O alvo recebe a nova senha por canal operacional seguro, fora desta pagina.
